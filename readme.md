# CS 283 - Systems Programming

Welcome to your personal CS 283 repository! This repository will contain all of your assignment work for the semester.

## Repository Structure

Each assignment will have its own directory:

```
cs283-yourusername/
├── README.md          (this file)
├── .gitignore
├── 00-Warmup/        (warmup assignment)
├── 01-Assignment/    (assignment 1)
├── 02-Assignment/    (assignment 2)
└── ...
```

## Getting Assignment Instructions and Starter Code

**All assignment instructions and starter code are located at:**

**https://github.com/drexel-systems/SysProg-Class/tree/main/assignments**

For each assignment:
1. Go to the link above and navigate to the appropriate assignment directory
2. Read the assignment instructions (README.md or PDF)
3. Download the starter code files
4. Create the assignment directory in this repository (if not already present)
5. Copy the starter files into the assignment directory
6. Complete the assignment
7. Commit and push your work to this repository

## Basic Git Workflow

```bash
# Navigate to your assignment directory
cd ~/cs283-yourusername/XX-Assignment

# Add your changes
git add .

# Commit with a meaningful message
git commit -m "Describe what you changed"

# Push to GitHub
git push origin main
```

## Important Reminders

- **Commit frequently** - Don't wait until the assignment is complete
- **Push regularly** - Your GitHub repository serves as a backup
- **Write good commit messages** - Future you will thank present you
- **Check Canvas** - Submission deadlines and additional instructions are posted there
